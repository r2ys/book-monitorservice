# 监控指标

监控指标从各个维度配置：

* 主机：
    * cpu使用率
    * 内存使用率
    * IO速率
    * 磁盘空间
* 服务
    * 响应时间
    * 错误率
    * 请求比例
    * 并发数
    * 延迟率
    * 事务处理
* 应用
    * 前端页面：响应时间、加载时间
    * 性能：响应时间、成功/失败数
    * 健康度：告警量
*
